# 使用mvvm+kotlin+jetpack搭建一个wanandroid客户端

## 涉及到的技术包括：
  * kotlin和协程
  * 依赖注入库使用koin
  * JetPack
    * databinding
    * livedata
    * viewmodel
    * lifecycle
    * room
    * viewPager2
  * MVVM架构
  * Repository设计
  * Retrofit作为网络库
  * glide图片加载
  * 第三方库(详见项目中的依赖)：
    * baseRecyclerViewAdapterHelper
    * banner
    * flowLayout
    * loading_helper
    * l_dialog
    * auto_size
    * noDrawable
